# 📚 Python Data Structures

Welcome to the **Data Structures** section of my **Basic Python** repository.

This folder contains beginner-friendly examples and practice programs to help you understand Python's built-in data structures. Each topic includes simple explanations and practical examples for learning and revision.

## 📖 Topics Covered

- 📋 List
  - Creating Lists
  - Indexing & Slicing
  - List Methods
  - List Operations

- 🔒 Tuple
  - Creating Tuples
  - Accessing Elements
  - Tuple Methods

- 📚 Dictionary
  - Key-Value Pairs
  - Dictionary Methods
  - Adding, Updating & Removing Data

- 🎯 Set
  - Creating Sets
  - Set Operations
  - Union
  - Intersection
  - Difference
  - Symmetric Difference
  - `issubset()`
  - `issuperset()`

- 🔤 String
  - String Indexing
  - String Slicing
  - String Methods
  - String Formatting

## 🎯 Learning Objectives

After completing this section, you will be able to:

- Understand Python's built-in data structures.
- Store and organize data efficiently.
- Perform common operations on Lists, Tuples, Dictionaries, Sets, and Strings.
- Choose the right data structure for different programming problems.
