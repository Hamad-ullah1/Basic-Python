# Python Functions

This repository contains my practice and learning materials on **Python Functions**. It includes examples, exercises, and explanations of different function concepts that are essential for writing clean, reusable, and modular Python code.

## 📖 Topics Covered

* Function Basics
* Defining and Calling Functions
* Parameters and Arguments
* Return Statement
* Default Parameters
* Keyword Arguments
* Variable-Length Arguments (`*args` and `**kwargs`)
* Lambda Functions
* Scope of Variables (Local & Global)
* Recursive Functions
* Built-in Functions
* User-Defined Functions
* Practical Function Examples

## 🎯 Learning Objectives

The purpose of this repository is to:

* Understand how functions improve code reusability.
* Learn to write clean and modular Python programs.
* Practice different types of function arguments.
* Explore advanced function concepts such as recursion and lambda expressions.
* Build a strong programming foundation for Machine Learning and Artificial Intelligence.
