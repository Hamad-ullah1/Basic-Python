# 🏗️ Object-Oriented Programming (OOPs) in Python

This repository contains my practice and learning material for **Object-Oriented Programming (OOP)** in Python. It includes beginner-friendly examples, exercises, and mini-projects covering the core OOP concepts.

The purpose of this repository is to strengthen my understanding of writing clean, reusable, and organized Python code using OOP principles.

---

## 📚 Topics Covered

- Classes and Objects
- Constructors (`__init__`)
- Instance Variables
- Instance Methods
- Class Variables
- Class Methods
- Static Methods
- Inheritance
- Method Overriding
- Abstract Classes
- Abstract Methods
- Dunder (Magic) Methods
  - `__init__`
  - `__str__`
  - `__repr__`
  - `__eq__`
  - `__add__`
- Object Comparison
- Operator Overloading

By completing these examples, I learned how to:
